Create form for upload image and add info for car.

== Description ==

Plugin "Car Form" allows You to add car's details directly to your site.

== Installation ==

To set the plugin up, you need to:

1. Upload `Car Form` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

That's all!
